## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Usage

1. Click "New Entry" to create a work journal entry
2. Fill in the date, title, and your notes
3. Your entries are automatically saved to local storage
4. View all your entries in chronological order
5. Delete entries by clicking the Ã— button

Perfect for tracking:
- Daily work activities
- Learnings and insights
- Meeting notes
- Code reviews
- Project progress
- Reflections
